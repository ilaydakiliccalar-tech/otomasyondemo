# Sanat Merkezi Randevu Takip Otomasyonu Prototipleri

Bu proje, bir Sanat Merkezi'nin randevu takip süreçlerini dijitalleştirmek amacıyla geliştirilen çok platformlu (Web, Mobil, Masaüstü) arayüz prototiplerini içermektedir.

## Proje İçeriği

### 1. Web Uygulaması (`web-app`)
*   **Teknoloji:** HTML5, CSS3, JavaScript.
*   **Özellikler:** Modern dashboard, randevu listesi, yeni randevu oluşturma modalı.
*   **Tasarım:** Modern, akıcı ve mobil uyumlu (responsive).

### 2. Mobil Uygulama (`mobile-app`)
*   **Teknoloji:** Android (Java/XML).
*   **Özellikler:** Kullanıcı dostu arayüz, hızlı randevu butonu, gelecek ders özeti.

### 3. Masaüstü Uygulaması (`desktop-app`)
*   **Teknoloji:** Java (Swing/Eclipse).
*   **Özellikler:** Yönetici paneli, eğitmen ve öğrenci yönetim tabloları.

## Kurulum ve Çalıştırma
*   **Web:** `web-app/index.html` dosyasını herhangi bir tarayıcıda açabilirsiniz.
*   **Masaüstü:** `MainFrame.java` dosyasını Eclipse veya herhangi bir Java IDE'sinde çalıştırabilirsiniz.
*   **Mobil:** `mobile-app` içerisindeki dosyaları bir Android Studio projesine dahil ederek inceleyebilirsiniz.
